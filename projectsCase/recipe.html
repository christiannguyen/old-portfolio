Used an online recipe API to generate a random recipe for the user to decide on. The API would supply the recipe ID, ingredients, link to the recipe, photo, and name. I grabbed this information using request calls, specifically AJAX.

This project introduced me to APIs and helped me understand how request calls worked. APIs are also really fun to play around with and I look forward to finding out interesting ones to work with! I didn't focus on the design or UI for this project specifically. It was one of my first projects that include AJAX requests and this helped familiarized me with AJAX requests and using JSON data.
