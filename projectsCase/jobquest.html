Created a website that would display job postings in an organized fashion. I developed and designed this project using create-react-app and AJAX calls from Indeed's API to gather their job posting data.

This was the first project where I used React to understand the basic fundamentals of it. This project included a lot more JavaScript coding and understanding of how React works. I appreciate a framework like React because it helped me understand what made those frameworks useful and how they can help with web development.
